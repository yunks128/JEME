## [1.0.0] - 2026-05-22

### Added
- JEME (JPL Earth Modeling Enterprise) dashboard for 8 science models: RAPID, CARDAMOM, CMS-Flux, ECCO, ISSM, MOMO-CHEM, LES, EDMF
- JEOE (JPL Earth Observation Enterprise) dashboard for NASA missions: GRACE, SWOT, TROPESS
- Per-model citation trend charts (annual and cumulative)
- Research domain analysis and geographic impact maps
- Engagement level classification (L1–L4)
- Cross-model network analysis: bridge papers, shared authors, connection matrix
- Model Capability Level (MCL) maturity assessment pages
- Uncertainty quantification pipeline (3-phase: deterministic scoring, LLM sampling, skeptic agent)
- Private Sector Engagement (PSE) explorer with D3 alumni network graph
- Multi-agent citation verification pipeline (Semantic Scholar, Crossref, keyword classifier)
- Peer-review filtering — only peer-reviewed publications included
- GitHub Pages deployment to `https://github.jpl.nasa.gov/pages/kyun/jeme/`
- SLIM open source compliance (CODE_OF_CONDUCT, GOVERNANCE, CHANGELOG, CITATION.cff, issue/PR templates)
