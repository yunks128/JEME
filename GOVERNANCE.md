# Governance

## Project Maintainers

The JEME Dashboard is maintained by the JPL Earth Science group. The primary maintainer is responsible for:

- Reviewing and merging pull requests
- Triaging issues
- Making release decisions
- Ensuring SLIM compliance

## Decision Making

- **Routine changes** (bug fixes, documentation, minor features): maintainer approval required.
- **Major changes** (new models/missions, architecture changes, data schema changes): discussion via GitHub issue before implementation.
- **Breaking changes**: must be documented in `CHANGELOG.md` and communicated in the PR description.

## Contributions

All contributions are made via pull requests. See the issue templates for reporting bugs or requesting features. PRs are reviewed on a best-effort basis.

## Releases

Releases are tagged on `main` following [Semantic Versioning](https://semver.org/) (`MAJOR.MINOR.PATCH`). Release notes are maintained in `CHANGELOG.md`.

## Contact

For questions not suited to a public issue, contact the maintainers through the JPL internal channels or open a GitHub issue.
